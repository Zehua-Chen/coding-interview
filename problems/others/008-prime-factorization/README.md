# 8 - Prime Factorizations

Write a function that returns the list of a number's prime factorization in ascending order

## Example

```
input: 1
output: []
```
```
input: 2
output: [2]
```
```
input: 64
output: [2, 2, 2, 2, 2, 2]
```
```
input: 315
output: [3, 3, 5, 7]
```

```
input: 997
output: [997]
```
